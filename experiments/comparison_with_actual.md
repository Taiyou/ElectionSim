## 8. 主な知見

### 精度
- 最も精度が高かったのは **v4b LLM全ペルソナ** (総誤差36席)
- LDP議席を最も正確に予測: **v8a キャリブレーションLLM** (予測247 vs 実際249, 差-2)
- 投票率を最も正確に予測: **v2 ルールベース** (予測55.76% vs 実際55.68%)

### 自民単独過半数の予測
- 2/6実験が正解
  - v2 ルールベース: X (不正解)
  - v4b LLM全ペルソナ: O (正解)
  - v8a キャリブレーションLLM: O (正解)
  - v9a ハイブリッド: X (不正解)
  - v10a 人口統計LLM: X (不正解)
  - v10b メモリ付きLLM: X (不正解)

### 体系的バイアス
- **全実験が中道改革連合の議席を大幅に過大予測**: 実際7席に対し、予測は22~69席
- **全実験がLDPの議席を過少予測**: 実際249席に対し、予測は179~252席
- 中道の過大予測は特にルールベース系・メモリ系で顕著 （v10b: +62席）

### 手法別の評価
- **キャリブレーションLLM (v8a)**: LDP予測が最も正確だが、中道も過大予測
- **LLM全ペルソナ (v4b)**: LDP予測は近いが、投票率が大幅に過大 (79.5%)
- **ルールベース (v2)**: 投票率は正確だが、LDP過少/中道過大のバイアスが大きい
- **ハイブリッド (v9a)**: ルールベースに近い結果、接戦区LLMの効果は限定的
- **人口統計/メモリ (v10a/b)**: 中道過大バイアスが最も大きい

### 教訓
1. LLMは中道・リベラル方向へのバイアスを持つ (全実験で確認)
2. キャリブレーション (v8a) が最もバイアスを緩和できた
3. ルールベースの支持率データ自体が2024年選挙ベースであり、2026年の自民大勝トレンドを反映できていなかった
4. 投票率予測はデカップリング方式 (v8a) で改善されたが、投票率と議席数の両方を正確に予測した実験はなかった
